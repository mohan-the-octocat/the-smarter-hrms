# 17.2 Definition of a Flexible Working Request

Definition of a Flexible Working Request A flexible working request under this policy generally means a request to do any or all of the following:
- Reduce or vary your working hours.
- Reduce or vary the days you work.
- Work from a different location (for example, from home).
- Any combination of the above arrangements.
